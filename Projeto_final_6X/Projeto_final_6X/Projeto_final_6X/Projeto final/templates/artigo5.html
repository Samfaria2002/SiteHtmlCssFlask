Extensões essenciais do Visual Studio Code

Por Samuel Faria

Aluno Centro Universitário UNA/Ciência da Computação

11/06 - 19:39

Auto Complete Tag

Todo programador já ficou ou sempre fica com uma pulga atrás da orelha e uma preguíça de sempre ter que fechar as tags do HTML de modo manual, felizmente, o Visual Studio Code nos permite baixar uma extensão para resolver esse problema. A Auto Complete Tag é um extensão que permite fechar de maneira automática as tags do HTML, basta baixar que ela é aplicada instantâneamente. Prático não é?

Color Picker/Color Info

Aqui temos uma dobradinha. De maneira resumida, ambas atuam de maneira a auxiliar o desenvolvedor a escolher e manejar as cores de seus projetos. A extensão Color Picker permite uma rápida e prática seleção de cores dentro do próprio programa, já o Color Info nos mostra a cor que escolhemos, o rgb, o nome ou o hexadecimal adquire o tom da cor que selecionamos. Ambas as extensões são muito boas separadas, mas juntas elas tornam o projeto mais organizado e prático.

Drácula

Quando comecei a usar o Visual Studio Code, algo me incomodava. Diferentemente do Sublime Text, as cores dos códigos eram brancos, dificultando um pouco a visualizar as propriedades e tags. Foi quando descobri a extensão Drácula; essa extensão permite alterar a cor das linhas de código, dando uma cor para cada função.

Live Server

De longe, a minha favorita. Sabe quando você está programando e sempre tem que dar um CTRL + S e reiniciar a página do navegador para atualizar o projeto? As vezes isso se torna meio improdutivo e cansativo. Instalando a extensão Live Server, seus projetos vão rodar em tempo real; apenas salvando o arquivo o resultado é atualizado automaticamente.

Lorem Ipsum

Essa é uma extensão super simples porém extremamente prática. Para quem não conhece, Lorem Ipsum é um texto em latim muito usado por desenvolvedores, a fim de testar seus campos de texto para ver se tudo ficou bem formatado. A extensão adiciona uma função de inserção automática de um Lorem Ipsum, basta definir uma tecla para ativar o texto.

Sobre

Bem-vindo ao nosso site criado para o projeto final da universidade UNA para os cursos das áreas da tecnologia da informação! Resolvemos criar um site sobre Web Design, mostrando e dando algumas observações acerca das tecnologias mais usadas no meio dos Web Designers, como o HTML, o CSS e softwares como o Photoshop.

Foi um projeto muito produtivo e divertido de se criar pois todos os membros do grupo aprenderam bastante, além de ajudar a criar boas amizades. Mesmo com algumas inexperiências na área da programação, nós ajudamos uns aos outros, o que tornou tudo muito menos complicado e mais tranquilo.
