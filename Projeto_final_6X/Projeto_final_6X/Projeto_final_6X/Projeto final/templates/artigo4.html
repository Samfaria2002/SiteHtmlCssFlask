UX e UI

Por Samuel Faria

Aluno Centro Universitário UNA/Ciência da Computação

14/06 - 14:05

UI

Mas o que significa? UI ou Interface de Usuário (User Interface), é toda a parte visual de uma aplicação, onde o usuário se conecta com o meio, seja um aplicativo, um site ou sistema. A interação pode ocorrer através de diversos elementos da interface, como um botão, texto ou imagens.
A User Interface possui padrões definidos de interface, como o Flat Design sendo um dos tipos de layout mais usados entre os profissionais UI sendo um estilo super comum na web, se caracterizando ao utilizar cores e formas chapadas com um estilo minimalista. O profissional de UI é responsável por criar toda a conexão direta com o usuário, além da interação e se preocupando com a Arquitetura da Informação.

UX

Ao se conectar com alguma aplicação, cada usuário se sente mais ou menos confortável ao a utilizar, é nisso que a UX ou Experiência do Usuário (User Experience) se baseia. É a área onde estudamos o lado emocional do usuário em relação a nossa interface, como o mesmo se sente ao utiliza-la. O profissional de UX se preocupa e tem como foco proporcionar a melhor exeriência possível aquele usuário diante a interface.

Sobre

Bem-vindo ao nosso site criado para o projeto final da universidade UNA para os cursos das áreas da tecnologia da informação! Resolvemos criar um site sobre Web Design, mostrando e dando algumas observações acerca das tecnologias mais usadas no meio dos Web Designers, como o HTML, o CSS e softwares como o Photoshop.

Foi um projeto muito produtivo e divertido de se criar pois todos os membros do grupo aprenderam bastante, além de ajudar a criar boas amizades. Mesmo com algumas inexperiências na área da programação, nós ajudamos uns aos outros, o que tornou tudo muito menos complicado e mais tranquilo.
