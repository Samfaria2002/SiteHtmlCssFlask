Sobre

Bem-vindo ao nosso site criado para o projeto final da universidade UNA para os cursos das áreas da tecnologia da informação! Resolvemos criar um site sobre Web Design, mostrando e dando algumas observações acerca das tecnologias mais usadas no meio dos Web Designers, como o HTML, o CSS e softwares como o Photoshop.

Foi um projeto muito produtivo e divertido de se criar pois todos os membros do grupo aprenderam bastante, além de ajudar a criar boas amizades. Mesmo com algumas inexperiências na área da programação, nós ajudamos uns aos outros, o que tornou tudo muito menos complicado e mais tranquilo.
